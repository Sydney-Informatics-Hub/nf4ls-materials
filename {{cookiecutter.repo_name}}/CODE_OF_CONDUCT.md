# Code of Conduct

This project follows the Australian BioCommons Code of Conduct published at:

https://zenodo.org/records/4276854

All contributors, maintainers, and participants are expected to adhere to the
standards of behaviour outlined in that Code of Conduct when engaging with this
repository and its associated activities.

## Scope

This Code of Conduct applies to all project spaces, including but not limited to:

- GitHub issues and pull requests
- GitHub discussions and reviews
- Project-related communications
- Training, workshops, and community interactions associated with these materials
